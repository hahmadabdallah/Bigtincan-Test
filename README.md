## Used technologies:
  <b> Frontend : </b>
    <ul>
    <li>React js</li>
    <li>Bootstap 4</li>
    <li>SASS</li>
    </ul>
  <b> Backend : </b>
    <ul>
    <li>JSON-Server: as fake API</li>
    </ul>
## Running Steps
   ### Frontend
    $ git clone https://github.com/hahmadabdallah/react-crud-file.git
    $ npm install
    $ npm start
    
   ### Backend
      $ cd server
      $ npm install
      $ json-server --watch db.json --port 3001
      
 ## Application Features:
   <ul>
    <li>Create file (upload from device or drag and drop)</li>
